# 📧 Automação de Envio de E-mails com Google Apps Script

Projeto desenvolvido para automatizar o envio de solicitações de contas via Gmail a partir do Google Sheets.

## 🚀 Funcionalidades
- Leitura automática da aba "Solicitações"
- Integração com aba "Frases"
- Envio automático com HTML
- Atualização de status para "Enviado"

## 🛠 Tecnologias
- Google Apps Script
- Google Sheets
- GmailApp
